## Como a IA foi utilizada

A IA atuou como **ferramenta de produtividade** sob direção do desenvolvedor. O ciclo de trabalho era:

1. **Humano** define requisitos (documentos `.md`, issues do GitHub, descrições de bugs)
2. **IA** lê a especificação, analisa o estado atual do código e implementa
3. **Humano** valida no browser, reporta problemas colando HTML/CSS/erros do console
4. **IA** corrige e itera até o resultado esperado
5. **Humano** controla versionamento (branches, commits, PRs)
